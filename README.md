# React Redux Toolkit Shop

A simple shop application created for learning Redux (Redux Toolkit).

## Technologies

- **React**
- **Redux Toolkit**
- **CSS Modules**

## How to run the project

1. Clone the repository:
   ```bash
   git clone https://github.com/SkoczekBoczek/react-redux-toolkit-shop.git
   ```
2. Enter the project directory and install dependencies:
   ```bash
   npm install
   ```
3. Run the application in development mode:
   ```bash
   npm start
   ```
